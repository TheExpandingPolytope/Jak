Polywolf

Download Source Examples

Polywolf is an embedded model viewer that runs entirely on javascript and WebGL 2.0. It is easy to use. Get started by downloading the latest build from github. Include it in your document, like so:

<script type='module'src="build/polywolf.js"></script>Then add this line of html:<div is='poly-wolf' url="PATH/TO/MODEL.gltf"></div> You have just created a polywolf model view. Now click the play button to view your 3d model.
