how to use

Polywolf is simple to use. Simply install the build source. Include it in your documenet.

<script type='module'src="build/polywolf.js"></script>. Then append this element to your document. <div is='poly-wolf' url="PATH/TO/MODEL.gltf"></div> The model is loaded through an XMLHttpRequest at the designated url. You can also set the height and width of the element. View Examples below for more.
